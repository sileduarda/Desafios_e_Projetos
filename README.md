# Desafios/Projetos

Olá! Seja bem-vindo ao meu repositório de Projetos desenvolvidos nos cursos de especialização que estou fazendo . :relaxed:
---

Este repositório a príncipio foi criado como parte de um desafio de Projeto do curso de GitHub da Digital Innovation One (DIO). O objetivo deste projeto era praticar e aprimorar habilidades em Git, GitHub e Markdown, além de colaborar com outros desenvolvedores. Diante disso, resolvi deixer esse repositório como um portfólio para demonstrar minhas habilidades em controle de versão, linguagem Markdown e análise de dados.

---
## Objetivos do Projeto
- Aprender a usar o Git e o GitHub de forma eficaz.
- Praticar a criação e edição de arquivos Markdown.
- Colaborar com outros desenvolvedores em um ambiente de controle de versão.
